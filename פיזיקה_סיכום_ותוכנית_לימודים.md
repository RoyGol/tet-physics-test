### נושא 1: אנרגיה מכנית — אנרגיית גובה ואנרגיית תנועה

#### תיאוריה
- **אנרגיית גובה (פוטנציאלית):** `Eₕ = mgh`
  - m = מסה (ק"ג), g = 10 N/kg, h = גובה (מטרים)
- **אנרגיית תנועה (קינטית):** `Eₖ = ½mv²`
  - m = מסה, v = מהירות (מ/ש)
- **אנרגיה מכנית כוללת:** `E_מכנית = Eₕ + Eₖ`
- **חוק שימור האנרגיה:** האנרגיה המכנית הכוללת קבועה לאורך תנועה ללא חיכוך
  - ⬆️ עולה בגובה → Eₕ גדלה, Eₖ קטנה
  - ⬇️ יורד → Eₕ קטנה, Eₖ גדלה

#### מושגים חשובים
- **משקל:** W = mg
- מישור ייחוס — גובה נמדד ביחס לנקודה שבוחרים (רצפה, שולחן, תקרה)
- כאשר אין חיכוך: אנרגיית גובה בראש = אנרגיית תנועה בתחתית

#### סוגי שאלות במבחן
- חישוב Eₕ ו-Eₖ בגבהים שונים
- מציאת מהירות בנקודה נתונה בעזרת חוק שימור האנרגיה
- ניתוח גרפים של שינויי אנרגיה כתלות בגובה
- השוואת מסלולים שונים (מגלשה, סכר, כדור)
- חישוב מהירות בתחתית מגלשה/סכר

#### דוגמת חישוב (מהמבחן)
```
מסה: 6,500 ק"ג, גובה: 20 מ'
Eₕ = 6500 × 10 × 20 = 1,300,000 ג'ול

בתחתית (h=0): כל האנרגיה הופכת לקינטית
½mv² = 1,300,000
v = √(2,600,000 / 6,500) = 20 מ/ש
```

### נושא 2: תחנה הידרואלקטרית — המרות אנרגיה

#### תיאוריה
- **רצף המרות האנרגיה:**
  ```
  אנרגיית גובה → אנרגיית תנועה → אנרגיית תנועה → אנרגיה חשמלית
  של המים          של המים         של הטורבינה      במחולל
  ```
- **נצילות:** `נצילות = (אנרגיה מופקת / אנרגיה מושקעת) × 100%`
- נצילות של ~90% = 90% הופכת לחשמל, 10% לחום ולאנרגיה לא שימושית

#### יתרונות וחסרונות של תחנה הידרואלקטרית

| יתרונות | חסרונות |
|---------|---------|
| מקור אנרגיה לא מתכלה | פגיעה במערכות אקולוגיות |
| ידידותית לסביבה (כמעט ללא גזי חממה) | סכנת שיטפונות בשבירת סכר |
| נצילות גבוהה (~90%) | עלות בנייה גבוהה |
| ניתן לוויסות מהיר של ייצור | תלות בתנאים גאוגרפיים |

#### סוגי שאלות במבחן
- זיהוי רצף המרות אנרגיה (שאלה סגורה)
- טיעון עם נימוק — יתרון + חיסרון
- חישוב אנרגיית גובה ומהירות מים
- ניתוח גרפים של 3 סוגי אנרגיה
- שאלות על נצילות

### נושא 3: מעגלים חשמליים

#### תיאוריה — מושגי יסוד
- **זרם חשמלי:** תנועה מכוונת של אלקטרונים חופשיים
- **עוצמת זרם:** `I = q/t` (אמפר = קולון/שנייה)
- **חוק אוהם:** `I = V/R`
  - I = עוצמת זרם (אמפר, A)
  - V = מתח (וולט, V)
  - R = התנגדות (אוהם, Ω)

#### סמלים בסכמת מעגל
| סמל | מה זה |
|-----|-------|
| ⊗ | נורה/מכשיר |
| A (עם עיגול) | אמפרמטר — מודד זרם |
| V (עם עיגול) | וולטמטר — מודד מתח |
| —‖— | סוללה/מקור מתח |

#### חיבור בטור vs. מקביל

| | חיבור בטור | חיבור במקביל |
|--|------------|--------------|
| **הזרם** | זהה בכל נקודה | שונה בכל ענף |
| **המתח** | מתחלק בין המכשירים | זהה בכל מכשיר = מתח המקור |
| **התנגדות כוללת** | R_כולל = R₁ + R₂ + R₃ | קטנה מהקטן שבהם |
| **תקלה במכשיר** | כל המעגל נפסק | שאר המכשירים ממשיכים לפעול |
| **שימוש** | נדיר בבית | **נפוץ בבית** |

#### חישובים
**חיבור מקביל:**
```
המתח על כל מכשיר = מתח המקור
I = V/R    (לכל מכשיר בנפרד)
דוגמה: V=220, R=50Ω → I = 220/50 = 4.4 A
```

**חיבור בטור:**
```
R_כולל = R₁ + R₂ + R₃
I_כולל = V_כולל / R_כולל
דוגמה: V=220, R₁=100, R₂=50, R₃=30
R_כולל = 180Ω → I = 220/180 = 1.22 A
```

### נושא 4: הספק וצריכת אנרגיה חשמלית

#### תיאוריה
- **הספק:** `P = E/t` = כמה אנרגיה נצרכת בשנייה
- **הספק חשמלי:** `P = V × I`
- יחידה: וואט (W) = ג'ול/שנייה
- **אנרגיה חשמלית:** `E_חשמלי = V × I × t`

#### עיקרון חשוב
- ככל שהספק **גבוה יותר** → צריכת **אנרגיה רבה יותר** באותו פרק זמן
- 1,200W > 2,200W? — לא! 2,200W צורך יותר (2.2 kW > 1.2 kW)
- השוואה: תמיד להמיר לאותן יחידות!

#### סוגי שאלות
- השוואת צריכת אנרגיה בין מכשירים
- חישוב זמן הפעלה

### נושא 5: אנרגיית חום (תרמית)

#### תיאוריה
- **אנרגיית חום:** `Q = m × c × ΔT`
  - m = מסה **בקילוגרמים** (שימו לב: לא גרמים!)
  - c = חום סגולי (של מים: **c = 4,200 J/(kg·°C)**)
  - ΔT = שינוי טמפרטורה = T_סופית − T_התחלתית

#### חיבור בין חום לחשמל
- כאשר נאמר "כל האנרגיה החשמלית מנוצלת לחימום":
  ```
  E_חשמלי = Q
  P × t = m × c × ΔT
  ```

#### דוגמת חישוב (מהמבחן)
```
נתונים: P=1200W, m=500g=0.5kg, T_התחלה=20°C, T_סוף=100°C
Q = 0.5 × 4200 × (100-20) = 0.5 × 4200 × 80 = 168,000 J
P × t = Q → 1200 × t = 168,000 → t = 140 שניות
```

## 💡 טיפים לטכניקת מבחן

1. **גרף** — תמיד: כותרת גרף + שם ציר + יחידות. בלי זה מורידים 5 נקודות.
2. **חישובים** — לכתוב נוסחה → הצבה → תוצאה עם יחידות. לא לדלג.
3. **שאלות פתוחות** — לכתוב **טענה** + **נימוק** (2 משפטים). טענה בלבד = חצי ניקוד.
4. **שאלת נצילות** — זכרי: אנרגיה מושקעת = 100%, מופקת = האחוז הנתון.
5. **חוק שימור** — אנרגיה לא נעלמת, רק מתמרת. בתחתית h=0 → כל Eₕ הפכה ל-Eₖ.
